# GrandMA2 Passport iOS

Нативный SwiftUI-перенос Android-версии.

## Что нужно для сборки

- Полный Xcode, не только Command Line Tools.
- iPhone с iOS 17+.
- Apple ID/Team для подписи на реальном iPhone.

## Как открыть

Открыть файл:

`ios_app/GrandMA2Passport/GrandMA2Passport.xcodeproj`

В Xcode выбрать свой Team в Signing & Capabilities и запустить на iPhone.

## Папка проектов

Приложение работает в своей iOS-песочнице:

`Documents/MA2_passports`

Через приложение "Файлы" папка будет доступна внутри документов приложения после установки.
