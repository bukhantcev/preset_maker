# Passport creator

## Subtitle
Паспорт спектакля из MA2 XML

## Description
Passport creator помогает быстро собрать паспорт света спектакля из XML-файла grandMA2.

Приложение читает шоу-файл, находит пресеты и приборы, создает проект, позволяет сфотографировать свет каждого прибора, добавить описание и автоматически собрать итоговые файлы паспорта. Также можно создать партитуру по cue-листу с выбранными полями.

Основные возможности:
- импорт XML-файла grandMA2;
- список проектов внутри приложения;
- съемка фото прибора прямо из карточки строки;
- добавление фото из галереи;
- описание для каждой записи;
- экспорт паспорта в XLSX и PDF;
- создание партитуры в XLSX и PDF;
- локальное хранение проектов и файлов.

Все данные хранятся на устройстве. Приложение не использует сервер, не отправляет файлы и не собирает аналитику.

## Review Notes
Приложение предназначено для светорежиссеров и художников по свету. Для проверки можно загрузить XML-файл grandMA2, создать проект, открыть раздел пресетов, сделать фото через камеру и убедиться, что в проекте создаются XLSX/PDF файлы.

Приложение работает локально. Учетная запись не требуется.
