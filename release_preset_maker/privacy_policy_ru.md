# Политика конфиденциальности

Дата вступления в силу: 8 июня 2026 г.

GrandMA2 Passport работает локально на устройстве пользователя и не собирает персональные данные.

## Какие данные обрабатывает приложение

Приложение может работать с файлами, выбранными пользователем:
- XML-файлы grandMA2;
- фотографии, сделанные камерой;
- фотографии, выбранные из галереи;
- созданные приложением XLSX и PDF файлы.

Эти данные используются только для создания паспорта спектакля и партитуры.

## Передача данных

Приложение не передает данные на серверы разработчика, не использует аналитику, рекламные SDK и трекинг.

Пользователь может самостоятельно отправить созданные файлы через системное меню iOS. В этом случае передача выполняется выбранным пользователем приложением или сервисом.

## Камера и фото

Доступ к камере используется только для съемки фото приборов. Доступ к галерее используется только для выбора фото, которое пользователь хочет добавить в проект.

## Хранение данных

Проекты и созданные файлы хранятся локально на устройстве в контейнере приложения и доступны пользователю через системные механизмы работы с файлами.
